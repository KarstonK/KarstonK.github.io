Getting Started:

fn main() {
    println!("Hello World");
}

Setting variables:

Example 1: let varName: varType = value; // will use the set variable type

Example 2: let varName = value; // will try to guess variable type

Integers:

Possible signed (can go negative) values: [i8, i16, i32, i64, i128, isize]

Possible unsigned (only 0 and positives) values: [u8, u16, u32, u64, u128, usize]

Example: let i = 2; // i32

Example: let i:u16 = 5; // u16

Floats:

Possible values: [f32, f64]

Example: let x = 1.2; // f64

Example: let y: f32 = 2.3; // f64

Booleans:

Possible values: bool

Example: let run = true;

Characters:

Possible values: char

Example: let f = 'f'; // char

Example: let f:char = 'f'; // char

Tuples:

Possible values: (varType, otherVarType, etc)

Example: let tup: (i32, u8, f32) = (-512, 1, 12.6); // i32, u8, f32

Arrays:

Possible values: [varType; varAmount]

Example: let nums = [1, 2, 3, 4, 5]; // type: i32 vars: infinity

Example: let nums:[u8; 5] = [1, 2, 3, 4, 5]; // type: u8 vars: 5

Eample: let nums = [1; 5]; // type: i32 vars: 5 and: all values are the number '1'

Maniputating strings:

.to_lowercase()

.to_uppercase()

.to_string()

Numeric Operations:

Addition: 3 + 8 // 11

Subtraction: 16.3 - 5.2 // 21.5

Multiplication: 7 * 4 // 48

division (quotient): 20.5 / 8.2 // 2.5

division (floored): 3 / 2 // 0

division (remainder): 37 % 5 // 2
