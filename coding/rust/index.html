setting variables:

Example 1: let varName: varType = value; // will use the set variable type

Example 2: let varName = value; // will try to guess variable type

Integer:

Signed (can go negative): [i8, i16, i32, i64, i128, isize]

Unsigned (only 0 and positives): [u8, u16, u32, u64, u128, usize]

Integer example: let i = 2; // i32

Integer example: let i:u16 = 5; // u16

Float: [f32, f64]

Float example: let x = 1.2; // f64

Float example: let y: f32 = 2.3; // f64

Boolean: bool

Boolean example: let run = true;

Character: char

Character example: let f = 'f'; // char

Character example: let f:char = 'f'; // char

Tuple: (varType, otherVarType, etc)

Tuple example: let tup: (i32, u8, f32) = (-512, 1, 12.6); // i32, u8, f32

Array: [varType; varAmount]

Array example: let nums = [1, 2, 3, 4, 5]; // type: i32 vars: infinity

Array example: let nums:[u8; 5] = [1, 2, 3, 4, 5]; // type: u8 vars: 5

Array example: let nums = [1; 5]; // type: i32 vars: 5 and: all values are the number '1'

Maniputating strings:

.to_lowercase()

.to_uppercase()

.to_string()

Numeric Operations:

Addition: 3 + 8 // 11

Subtraction: 16.3 - 5.2 // 21.5

Multiplication: 7 * 4 // 48

division (quotient): 20.5 / 8.2 // 2.5

division (floored): 3 / 2 // 0

division (remainder): 37 % 5 // 2
